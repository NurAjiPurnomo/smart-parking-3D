# Smart Parking 3D

## Deskripsi

Smart Parking 3D merupakan aplikasi simulasi parkir berbasis Unity yang dirancang untuk memberikan pengalaman visual interaktif dalam pengelolaan area parkir. Proyek ini dikembangkan sebagai media pembelajaran dan pengembangan keterampilan dalam bidang pengembangan game 3D, simulasi, dan sistem parkir cerdas.

## Fitur Utama

* Lingkungan 3D interaktif
* Simulasi kendaraan dan area parkir
* Kontrol kendaraan yang responsif
* Tampilan lingkungan kota (city environment)
* Mendukung pengembangan dan pengujian konsep smart parking

## Teknologi yang Digunakan

* Unity
* C#
* Git & GitHub
